# Notes Content Guidelines

## Socratic Tutoring Notes

A note style based on transcripts of exploratory dialogues with Claude Code (local), rephrased for readability.

**Source**: Claude Code CLI session transcript, cleaned up — not raw chat logs.

**Format**:
- Follows the question-driven exploration structure: start with an initial summary or framing, then ask questions to probe deeper, selectively expanding on chosen threads
- Rephrased for clarity and conciseness, but preserves the question-and-answer progression
- Use headers or bold questions to make the inquiry structure visible to the reader
- Concise answers — no padding, no unnecessary prose

**When to use**: When exploring a topic through dialogue produces something worth preserving — the question trail itself is the value, not just the conclusions.

**Tone**: Direct, curious, educational. Reads like a thinking-out-loud exploration, not a polished essay.
